# 🎮 Forum App

Aplikasi forum sederhana untuk komunitas gamer kecil.

## 🚀 Tech Stack
- **Backend**: Node.js, Express, MySQL  
- **Frontend**: React (Vite)

## ✨ Fitur
- Register & Login (menggunakan JWT Authentication)
- Membuat Postingan
- Memberikan Komentar

## 📦 Cara Menjalankan Project

### 🔧 Backend
Masuk ke folder backend:
```bash
cd backend

```Install dependencies:
npm install

```Jalankan server:
node index.js

```Server akan berjalan di:
http://localhost:3000

```🎨 Frontend
Masuk ke folder frontend:

cd frontend

Install dependencies:
npm install

Jalankan aplikasi:
npm run dev

Frontend akan berjalan di:
http://localhost:5173
